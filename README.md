## Plan

### Detailed Plan
1. 평가 지표 기준 정하기 -> 교육, 안전, 공공시설
2. 교육 -> 학교, 학원 데이터 + 진학률 상관관계 분석
3. 안전 -> 범죄, 사고, 유흥시설 분석
4. 공공시설 -> 생활인구 대비 공공시설(도서관,공원) 분석
5. 의료 -> 생활인구 대비 소아과 비율 분석
6. 주거비 -> 주거지 유형 별 가격대 분석
